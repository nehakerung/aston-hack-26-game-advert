# Game Advertisement for Untold Studios

An interactive mini-game designed to boost audience engagement with advertising campaigns. This demo showcases Untold Studios' recent work with Octopus Energy for BMW's campaign.

## Overview

This playable advertisement transforms passive viewing into active participation. The game is fully customizable to match any campaign theme - for example, a Waitrose space-themed ad would feature a space-themed game.

![screenshot of home](documents/home.png)


## Technology Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Features

- YouTube video integration with autoplay
- Responsive design
- Smooth animations and transitions
- Direct play-through functionality
